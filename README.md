# MO Legislator Web Scraper

Created for [100 Days of Code: The Complete Python Bootcamp](https://www.udemy.com/course/100-days-of-code)

Scrapes the websites for the Missouri state House and Senate to get contact information and committee assignments for
members.

The Missouri House scraping uses Beautiful Soup, because they kindly provide XML files specifically for scrapers.
The Missouri Senate scraping uses Selenium to navigate through the site and extract the relevant information from the
webpages.

The final output is an XSLX file with multiple sheets.
